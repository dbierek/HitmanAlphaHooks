# DInput8Proxy
This project adds some hooks to run functions from the Hitman 2015 Alpha code

## Usage and Debugging
1. Open the solution in Visual Studio
1. Build this lib
2. Put dinput8.dll in the Hitman Alpha folder (e.g. D:\HitmanAlpha\Hitman IO)
